
# 🚀 Nexus-H7 FD Master Node
### High-Performance Motion Controller for OpenPnP / grblHAL

**Nexus-H7 FD** — это высокоскоростная платформа управления движением, разработанная для профессиональных SMT-установщиков. Построенная на базе **STM32H723 (550 МГц)**, она реализует распределенную архитектуру управления через двойную шину **CAN FD**, что устраняет сложность проводки и максимизирует частоту генерации шагов.

---

## 🛠 Технические характеристики (Technical Specifications)

*   **MCU:** STM32H723VGT6, ARM Cortex-M7 @ **550MHz**.
*   **Firmware:** Полная совместимость с **grblHAL** (H7 driver fixes by dresco).
*   **Step/Dir Performance:** Частота импульсов до **800 KГц** для локальных осей X/Y.
*   **Изолированные интерфейсы связи:**
    *   **Dual CAN FD:** 2 порта (TD541SCANFD) для распределенных узлов (фидеры, голова, актуаторы).
    *   **RS-485:** 1 порт (TDH541S485H) для Modbus или панелей Nextion HMI.

### Периферия на борту:
*   **Оси:** 2 локальных (X/Y) + до 6 дополнительных через CAN FD.
*   **Память:** FRAM FM24CL16B (I2C) для энергонезависимого хранения параметров.
*   **Входы:** 6 буферизованных входов (74LVC) для E-Stop, Run, Alarm, Limits.
*   **Выходы:** 4 сильноточных PWM MOSFET (NTMFS5C628) с изолированными драйверами UCC21520.
*   **Аналоговые входы:** 2x ADC для вакуумных сенсоров (XGZP6847), 2x ADC для джойстика,2x ADC в резерве.

---

## 🔌 Программная интеграция (Software Integration)

*   **Core:** Верифицирован драйвер grblHAL STM32H7xx.
*   **Plugin:** Исправлена интеграция плагина OpenPnP.
*   **Key Fix:** Маппинг ADC для PA0/PA1 корректно определен как **Aux analog 0/1 (E0/E1)**. Это позволяет получать прямую обратную связь от сенсоров вакуума в актуаторах OpenPnP.

---

## 🏗 Распределенная архитектура (The Nexus Ecosystem)

Система спроектирована для работы с узлами **Nexus-Tower** и **Nexus-Drive**:
*   **Протокол:** Кастомный набор высокоскоростных сообщений CAN FD.
*   **Slave MCU:** STM32G431 (FDCAN, поддержка FOC).
*   **Функции:** Локальное управление моторами (Servo/Stepper), измерение вакуума и управление клапанами непосредственно «в точке использования».
