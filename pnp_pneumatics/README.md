## ⌨️ Управление через G-code (M-команды)

Для управления адаптивным вакуумным ресивером через консоль **OpenPnP**, **G-code sender** или терминал используется кастомная команда `M100`.

### Справочник команд (M-codes)


| Команда          | Описание           | Пример          | Пояснение |
| **M100 V[n]**    | Установка вакуума  | `M100 V-25`     | Установить разрежение -25 кПа. Насос плавно разгонится. |
| **M100 P[n]**    | Установка давления | `M100 P15`      | Установить избыточное давление 15 кПа для сброса. |
| **M100 V0 P0**   | Полная остановка   | `M100 V0 P0`    | Целевые значения — ноль. Оба двигателя плавно остановятся. |
| **M100 V.. P..** | Комбинированная    | `M100 V-30 P10` | Одновременная настройка обоих контуров. |

## 📊 Рекомендованные параметры вакуума и сброса

Оптимальный уровень разрежения зависит от веса компонента, площади его поверхности и типа сопла. Правильная настройка предотвращает повреждение мелких деталей и потерю тяжелых чипов при перемещении головы.

### Таблица настроек по типам компонентов



| Тип компонента | Вакуум (V, кПа) | Сброс (P, кПа) | Пояснение |
| :--- | :--- | :--- | :--- |
| **0201 / 0402** | `-5 ... -10` | `+2 ... +5` | Минимальное давление, чтобы деталь не затянуло в сопло. |
| **0603 / 0805** | `-15 ... -20` | `+5 ... +8` | Стандартные значения для большинства пассивных элементов. |
| **SOT-23 / SOIC** | `-25 ... -30` | `+8 ... +12` | Надежный захват для компенсации инерции при движении. |
| **TQFP / QFN** | `-35 ... -45` | `+12 ... +15` | Тяжелые чипы. Требуется максимальное прилегание. |
| **Разъемы / Габарит**| `-50 ... -65` | `+15 ... +20` | Максимальная мощность для тяжелых деталей. |

## 🛠 Тонкая настройка режимов

### 1. Мягкий старт (0201/0402)
Используйте `M100 V-7`. Низкое разрежение позволяет соплу аккуратно «прикоснуться» к детали, не выбивая её из ячейки ленты воздушным потоком.

### 2. Эффект Blow-off (Сброс)
Давление `P` в ресивере должно компенсировать остаточный вакуум в трубках:
*   **Деталь подпрыгивает?** — Уменьшите значение `P`.
*   **Деталь липнет к соплу?** — Увеличьте значение `P`.

### 3. Адаптивный порог (Vacuum Sensing)
В настройках OpenPnP установите порог срабатывания (**Threshold**) на **20-30% ниже** целевого значения.
> *Пример: Если `V-25`, фиксируйте ошибку захвата (Part Off) при падении вакуума до `-18...-20` кПа.*

##  Быстрый тест через консоль (G-code Console)

Для ручной проверки выполните последовательность:
1. `M100 V-20` — (Ждем 1 сек) — Проверка захвата 0805.
2. `M100 V0 P10` — (Ждем 0.2 сек) — Сброс детали.
3. `M100 P0` — Остановка компрессора.

> **Совет:** Сохраните эти значения в настройках **Feeders** или **Nozzles** в OpenPnP для автоматического применения при каждой смене компонента.
